# Medical Voice Assistant

> Telegram-бот, который слушает пациентов и помогает врачам разобраться

Пациент записывает голосовое сообщение про своё состояние — бот распознаёт речь, анализирует жалобу и даёт врачу структурированную сводку с рекомендациями.

## Что умеет

- Распознаёт медицинскую речь через SaluteSpeech
- Разбирает обращение через Groq AI
- Определяет тип проблемы и срочность
- Выдаёт понятный отчёт для врача
- Работает быстро

## Установка

**1. Поставьте библиотеки:**
```bash
pip install -r requirements.txt
```

**2. Создайте `.env`:**
```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
GROQ_API_KEY=ваш_ключ_groq
SALUTE_SPEECH_AUTH_KEY=ваш_токен_salutespeech
```

Токены:
- Telegram — [@BotFather](https://t.me/botfather)
- Groq — [console.groq.com](https://console.groq.com)
- SaluteSpeech — по доступу к API

**3. Запустите:**
```bash
python main.py
```

## Как работает

1. Пациент отправляет голосовое: "Болит голова уже неделю"
2. Бот распознаёт речь
3. Анализирует жалобу
4. Врач получает отчёт: симптомы, возможные причины, что спросить, что назначить

## Примеры обращений

- "У меня давление скачет после еды"
- "Как лечить простуду?"
- "Аллергия после таблеток"
- "Температура третий день держится"
